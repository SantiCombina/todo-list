# ToDo List

Project created on [Vite](https://vitejs.dev/) and [React](https://reactjs.org/) with [Tailwind CSS](https://tailwindcss.com/).

## Project setup

1- Clone the repository

2- Install dependencies

```
npm install
```

3- Run the project

```
npm run dev
```
